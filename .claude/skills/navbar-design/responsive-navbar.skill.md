# Responsive Navbar Design Skill

Creates a responsive navigation bar with mobile menu functionality using modern CSS and JavaScript techniques.

## Purpose
Generate a responsive navbar that adapts to different screen sizes with a collapsible mobile menu for smaller devices.

## Implementation Steps

1. Create HTML structure with semantic navigation elements
2. Implement CSS with flexbox or grid for desktop layout
3. Add media queries for mobile responsiveness
4. Create hamburger menu icon and animation
5. Implement JavaScript toggle functionality for mobile menu
6. Ensure accessibility with proper ARIA attributes

## Features to Include

- Hamburger menu icon that transforms to X when active
- Smooth slide-in/slide-out animation for mobile menu
- Proper focus management for keyboard navigation
- Semantic HTML with `<nav>` element
- ARIA attributes for screen readers
- Mobile-first CSS approach
- Hover effects for desktop users
- Dropdown support for submenu items (optional)

## Technology Stack
- HTML5 semantic elements
- CSS3 (Flexbox/Grid, Transitions, Media Queries)
- Vanilla JavaScript or minimal framework code
- Accessibility best practices

## Files to Generate
- HTML structure (if needed)
- CSS stylesheet with responsive styles
- JavaScript for toggle functionality
- Optional: Component file if using React/Vue/Svelte

## Customization Options
- Color scheme
- Menu items
- Logo placement
- Animation preferences
- Breakpoint values
